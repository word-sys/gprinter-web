# gprinter-web
GPrinter Devices Supported Print Application Web-Based, requires gprinter-linux-desktop to work

<img src="https://github.com/word-sys/gprinter-web/blob/main/screenshots.png" width="800" height="600"/>

# How To Use:
## Requirements:
Flask(for web), pillow(for drawing), pymupdf(for pdf drawing), gprinter-linux-desktop(for required files, used as library: https://github.com/word-sys/gprinter-linux-desktop.git, use on same directory: ex. /work/gprinter-linux-desktop and /work/gprinter-web [same directory])

``` bash
python3 server.py
```

Use localhost to access website: http://localhost:5001
